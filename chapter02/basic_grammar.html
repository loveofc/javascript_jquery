2.1 기본 용어

    2.1.1 표현식과 문장

  • "값"을 만들어내는 간단한 코드를 "표현식"이라고 부른다.

    • 273

    10 + 20 + 30 * 2

    'Hello World'

  • 하나 이상의 표현식이 모여 "문장"이라고 부른다.
  • 하나의 표현식도 ';'세미콜론 찍히면 문장이라 부른다.

    • 10+20+30+*2;

    var rintiantta = "Rint'+'Ian'+Tta';

    273;

    2.1.2 키워드

  • 자바스크립트가 처음 만들어질 때 정해진 특별한 의미를 가진 단어를 '키워드'라고 한다.
  • 키워드는 변수로 사용할 수 없다.
    • break else return if ......
    new null in do ......
    try typeof throw case ......

    2.1.3 식별자

  • 식별자는 변수명과 함수명을 구별하기 위해 사용한다
  • 네 가지 규칙만 만족한다면 모든 안어를 식별자로 사용할 수 있다.
    • 키워드를 사용하면 안된다.
    • 숫자로 시작하면 안된다.
    • 특수 문자는 _과 $만 허용된다.
    • 공백 문자를 포함할 수 없다.

    • alpha

    alpha10

    _alpha

    $alpha

    AlPha

  • 다만 키워드, 숫자로 시작, 공백 문자가 있으면 안된다.

    • break

    273asd

    has space

  • 식별자 중 변수, 함수, 메서드, 속성 구분

    • alert("Hello World") - 함수

    input - 변수

    Math.PI - 속성

    Math.abs(-200) - 메서드

    2.1.4 주석

  • 자바스크립트에서 주석방법은 "//" 한줄, "/**/"여러줄 이렇게 두가지 바법이 있다.
